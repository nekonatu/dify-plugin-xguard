# Privacy Policy — XGuard Plugin

## Data Handling

This plugin sends user input text to a self-hosted XGuard service for content safety detection. No data is sent to any third-party or external service.

## What data is transmitted

- The text content submitted for safety checking

## Where data is sent

Data is sent only to the XGuard service URL configured by the administrator. This is typically a local or internal network address.

## Data storage

This plugin does not store any data. The XGuard service performs stateless inference and does not persist request data.

## Third-party services

None. All processing happens on your own infrastructure.
